# Ethical Compliance Checklist

## 1. Data Integrity
- [ ] All logs contain timestamps
- [ ] No unverified data sources
- [ ] Companion actions have human witness entries

## 2. Privacy Assurance
- [ ] No collection of personal identifiers
- [ ] No off-device transmissions without attestation

## 3. Behavior Oversight
- [ ] Companion can say "I don't know"
- [ ] All generated insights reviewed by human operators

## 4. Publication Ethics
- [ ] Attribution and contributor credits intact
- [ ] No selective data omission
- [ ] Shared under CC BY-SA 4.0
